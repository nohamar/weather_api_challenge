# Weather API Challenge

A React + Vite weather dashboard that allows users to search for cities and view current weather conditions and forecast data.

The application uses the GeoDB Cities API to search for cities and retrieve their coordinates, then uses the OpenWeatherMap API to fetch current weather and forecast information based on latitude and longitude.

## Features

- City search with async dropdown suggestions
- Current weather display
- Forecast display
- Loading and error handling
- Current location weather using the browser Geolocation API
- Clear search input when using current location

## Technologies Used

- React
- Vite
- JavaScript
- OpenWeatherMap API
- GeoDB Cities API
- react-select-async-paginate
- react-accessible-accordion
